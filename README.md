# [PT-BR]
### Descrição do Projeto
Esse projeto é um protótipo de lading page para uma academia da Praia Grande - SP chamada Bravus.

### Tecnologias Utilizadas
- Nextjs
- Typescript

### Como rodar esse projeto?👨‍💻
- Clone o repositório: ```git clone https://github.com/davirandos/bravus-bjj.git```
- No terminal, rode ```npm install``` ou ```npm i```
- Rode ```npm run dev```
- ***Clonado!***

### Pastas do Projeto
- ```./src/components```: componentes da página
- ```./public```: local das fotos utilizadas

### Funcionalidades
- Grade de Aulas (Tabela completa e simulada)
- Carrossel de fotos de treinos
- Localização por Google Maps
- Redes Sociais
- Formulário de Contato
